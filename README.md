# react-native-tools
React Native bill of materials

## Install

```sh
npm i --save-dev gblejman/react-native-tools
```

## Displaying lint output in the editor

Include this line in `package.json`:

```js
{
  // ...
  "eslintConfig": {
    "extends": "./node_modules/react-native-tools/eslint.js"
  }
}
```
